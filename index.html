Git

  • git clone url_to_clone: creates a git directory on the local machine of the github repository at the url
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all modified files in the working branch to be committed
  • git commit -m: commits all added files, -m means with a message, like a comment
  • git pull origin main: pulls down all changes to local machine. Replace main with something else to narrow scope
  • git push origin branch: pushes up changes in "branch," replace "branch" with the branch name
  • More to be added to list later
